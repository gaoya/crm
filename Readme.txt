项目设置模式：
    设置一个页面展示数据的表
    T_Param
    id id 编号
    p_id 数据id
    p_name 数据name
    p_placeholder 数据提示
    p_label  标签
    p_list Y-0 N-1 0-显示 1-不显示
    p_list_lay_data list中lay-data= 数据的设置 Json 数据
    p_add  Y-0 N-1 0-显示 1-不显示
    p_update Y-0 N-1 0-显示 1-不显示
    p_del Y-0 N-1 0-显示 1-不显示
    p_search Y-0 N-1 0 显示 1-不显示
    p_type 数据类型
    p_params 数据参数值（默认值） Json 数据
    p_user 创建人
    p_time 创建时间
    p_remark 备注
